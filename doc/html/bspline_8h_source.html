101  // Methods implementing the recursive algorithm for computing the basis
102  // matrix per Theorem 1 of "General Matrix Representations for B-Splines".
103  // M_k = [ M_km1 ] * A + [ 0^T ] * B
104  // = [ 0^T ] [ M_km1 ]
105  //
106  // A = [ 1-d_0,i-k+2 d_0,i-k+2 ]
107  // [ 1-d_0,i-k+3 d_0,i-k+3 ]
108  // [ ... ]

111  // B = [ -d_1,i-k+2 d_1,i-k+2 ]
112  // [ -d_1,i-k+3 d_0,i-k+3 ]
113  // [ ... ]
114  // [ -d_1,i d_1,i ]
115  // where k is the spline order and i is the spline segment number.
